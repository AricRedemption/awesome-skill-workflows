# Skill Wiki Hot Coverage 2026

## Goal

Push `skills/wiki/` beyond a few clusters into a broader public-marketplace
coverage surface using three final-page types:

- representative individual skill pages
- category hub pages
- industry hub pages

## Why This Expansion

The first enrichment pass solved the "too empty" problem for direct search hits,
but query coverage was still strongest for engineering and generic office terms.
The next highest-signal gap was browse-style discovery:

- category queries such as `security`, `communication`, or `collaboration`
- industry queries such as `healthcare`, `education`, or `game dev`
- featured-skill queries coming from public registries

## Page Types Added

1. Open Agent Skills featured skill pages
2. Open Agent Skills category hubs
3. Skills Hub industry hubs

## Evidence Basis

- Open Agent Skills homepage visible category counts and featured skills
- Skills Hub homepage visible domain and industry categories

## Boundary

These pages are final landed Skill Wiki pages for search and discovery. They are
not runtime-certified workflow proofs.
