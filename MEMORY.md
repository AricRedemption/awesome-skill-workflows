# Project Memory

This file is reserved for durable project-local preferences and facts that
should affect future work in this repository.

Use `MEMORY.md` for:

- stable user preferences that apply to this project,
- durable project decisions,
- long-lived terminology,
- repeated workflow expectations that are not already owned by `AGENTS.md`.

Do not use this file for:

- run-specific evidence,
- raw conversation logs,
- sensitive values,
- cookies, tokens, browser profile paths, or credentials.

Memory updates should be made through an approved draft under
`evolution-drafts/`. Evidence-backed workflow lessons should usually start in
`evolution/` or `workflow-kb/` before becoming project memory.
