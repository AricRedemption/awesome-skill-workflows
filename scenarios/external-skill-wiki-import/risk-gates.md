# External Skill Wiki Import Risk Gates

## Required Gates

- `public_source_gate`: passed
- `boundary_honesty_gate`: passed
- `searchability_gate`: passed
- `no_false_promotion_gate`: passed

## Notes

- This scenario does not grant permission for autonomous external actions.
- This scenario does not certify the imported skill's runtime behavior.
- This scenario only certifies that the source is suitable for wiki-level
  discovery and reference.
