# External Skill Wiki Import Scoring Rubric

## Pass Conditions

A source passes this scenario when all of the following are true:

1. the source repo is public and reachable,
2. the imported wiki page records source repo, URL, and import status,
3. the page explains when to use and when not to use the source,
4. `find-skill` can retrieve the page using at least one natural query,
5. the page does not claim local functional verification or promotion.

## Failure Conditions

- missing provenance
- missing search proof
- imported page misrepresented as fully promoted or locally verified
- no clear scope / non-scope boundary
